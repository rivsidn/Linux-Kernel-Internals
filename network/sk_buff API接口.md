



```bash
  +-------------------------------------------------------------+
  |                                                             |
  +-------------------------------------------------------------+
  ^         ^                                       ^           ^
  |         |  <----------------------------------> |           |
  |         |                len                    |           |

head      data                                     tail        end
    <------  ------>                                 ------->
  skb_push() skb_pull()                              skb_put()
```





| 函数         | 作用                     |
| ------------ | ------------------------ |
| skb_put      | 尾部填充数据，移动`tail` |
| skb_push     | 头部填充数据，移动`data` |
| skb_pull     | 头部移除数据，移动`data` |
| skb_headroom | data-head                |
| skb_tailroom | end-tail                 |
| skb_reserve  | 保留一部分数据           |
| skb_trim     | 数据截取                 |
