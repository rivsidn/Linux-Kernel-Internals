假设虚拟机名称为`ubuntu`。



| 命令                 | 说明                            |
| -------------------- | ------------------------------- |
| virsh list [--all]   | 查看当前运行/安装的所有有虚拟机 |
|                      |                                 |
| virsh edit ubutun    | 编辑虚拟机配置文件              |
|                      |                                 |
| virsh destroy ubuntu | 关闭虚拟机                      |
| virsh start ubuntu   | 启动虚拟机                      |
|                      |                                 |
|                      |                                 |
